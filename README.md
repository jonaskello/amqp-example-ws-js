# amqp-example-ws-js

Example of @cloudamqp/amqp-client package connecting to RabbitMQ/AMQP via websockets in a browser application using javascript.

## How to run

```bash
# install deps
yarn
# start rabbit and websockets relay
docker compose up -d
# start the app
yarn start
```

Open http://localhost:8080 in a web browser.
